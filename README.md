## [EN] Description
Visualization of a binary tree in JavaScript.

## Features
* Support for nodes in numeric and text format;
* Automatic tree balancing;
* Animated visualization of operations on a tree;
* Delete by name and by node ID;
* Duplicate nodes;
* Save and load.
